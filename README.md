# powerpoint-generator

A Python tool that generates PowerPoint presentations from a plain-text template, with synthetic student data generation for testing.

## The Problem

- Creating slide decks with repeated or data-driven content by hand is slow and error-prone.
- A programmatic approach allows slides to be generated consistently from a structured template, useful for reports or standardised presentations.

## The Approach

**Inputs:** A plain-text slide template (`src/presentation_content_template.txt`) defining slide titles, content, and layout markers. Optionally, a synthetic CSV of student data generated by `src/data_generator.py`.

**Processing:** `src/powerpoint_generator.py` reads the template file, parses slide definitions, and uses `python-pptx` to construct a `.pptx` file. `data_generator.py` creates realistic fake student records across engineering disciplines for use in testing or demonstration slides.

**Outputs:** A populated `.pptx` presentation file.

## Value Delivered

- Reduces slide creation time for reports with structured, repeating content.
- Template-driven approach means non-developers can control content without touching Python code.
- Synthetic data generator supports testing without using real data.

## Scope & Status

- **Project type:** Internal utility / learning tool
- **Current state:** Archived
- **Known limitations:**
  - Template format is custom and not validated; malformed templates will cause errors.
  - Limited slide layout support; complex visual formatting is not handled.
  - No tests implemented.

## Tech Stack

- **Language:** Python
- **Library:** python-pptx
- **Input format:** Plain text template

## Who This Is For

Developers or analysts who need to automate generation of structured slide decks from templates.

## Getting Started

```bash
# Install dependencies
pip install python-pptx

# Generate a presentation from the template
python src/powerpoint_generator.py
```
